# insplanet official Front-end Document

## CI/CD
### vercel로 자동 배포
 - 기존 가비아 컨테이너 호스팅을 사용하는 경우, 클라이언트사이드 라우팅에 대한 대응이 되지 않아 404 오류 페이지 발생.
 - 기존 원본 코드와 배포 코드의 저장소가 분리되어 유지보수의 어려움.
